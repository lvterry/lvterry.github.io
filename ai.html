AI Usage

I use AI as leverage, not as a ghostwriter.

I use ChatGPT for research and brainstorm. In some way, it replaced my use of traditional search engines like Google.

I don't use them to generate articles that are published on this site. Everything on this site is written by me, not an AI.

I use Claude Code and OpenAI Codex for coding and debugging.

If my usage changes in the future, I will update this page.
